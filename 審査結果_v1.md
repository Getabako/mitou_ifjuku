## 修正提案（優先度順）

### 🔴 S（致命的：これがないと確実に落ちる）

#### S-1: 予算内訳の作成
**現状**: 予算内訳が存在しない
**必要な修正**: 公募要領⑧に従い、以下を表形式で作成
```
| メンバー | 月 | 作業時間(h) | 単価 | 金額 |
例：高崎 | 7月 | 160h | 5,000円 | 800,000円 |
```
- 1名上限 800万円、プロジェクト上限 1,600万円
- 1人あたり月200時間を超えないこと
- **影響**: C-5 が 0→4点（+20点分の影響）

#### S-2: 競合分析の追加
**現状**: 競合分析が一切ない
**必要な修正**: 以下の競合との比較表を作成
- Minecraft Education Edition（ゲーム×教育の大手）
- LITALICOワンダー（発達障害×IT教育の国内最大手）
- Roblox Education（メタバース×教育）
- CogCam / Cognoa等（AI×発達スクリーニング）

比較軸：①アセスメント機能 ②個別最適化 ③経済圏接続 ④対象ユーザー
- **影響**: A-4, B-4 が全員 1→4点（+30点分の影響）

#### S-3: アセスメントアルゴリズムの技術的記述
**現状**: 「マウス軌跡から視覚運動協応を特定する」程度の記述
**必要な修正**:
```
行動指標 → 特徴量 → 推論モデル → WAISスコアとの対応
例：
- マウス軌跡の曲率分散 → DCDスクリーニングスコアとの相関（r=0.7, 先行研究: XX, 2023）
- クリック間隔の変動係数 → ADHD-RSスコアとの相関（仮説）
- 建築物の左右対称性 → 空間認知能力（WAIS-IV 積木模様の代替指標）
```
少なくとも「仮説レベル」でも、**どの行動データがどの認知指標と対応するか**の具体的なマッピングが必須
- **影響**: A-1, C-2 が大幅加点（+15点分の影響）

### 🟠 A（重要：採点に大きく影響）

#### A-1: 市場規模の数値化
**修正案**:
```
TAM: 教育市場 4.8兆円
SAM: 不登校・発達障害向け教育市場 約3,000億円
  - 不登校児童生徒数 約30万人（文科省2024年調査）
  - 発達障害の可能性がある児童 約8.8%（文科省2022年調査）
  - フリースクール市場 約300億円
SOM: 初年度 秋田県内 50名 × 月額3万円 = 1,800万円/年
```

#### A-2: MVPの明確な定義
**現状**: Phase 1-3の粒度が粗い
**修正案**:
```
MVP（8ヶ月で「これだけは完成させる」もの）：
1. Stealth WAISエンジン v1（操作ログからWAIS 4指標を推定するアルゴリズム）
2. クエスト3種（PSI/WMI/Modality測定用）
3. アセスメントレポート自動生成機能

NOT MVP（事業期間後に開発）：
- ブロックチェーン/NFT → 削除
- DAO機能 → 削除
- 企業マッチング → Phase 3で「設計のみ」
```

#### A-3: 倫理面・プライバシーへの言及
**必要な修正**:
- 子供のデータを扱うことへの倫理的配慮
- 保護者インフォームドコンセントのプロセス
- データの匿名化・暗号化・保管方法
- if(塾)での実証実験における倫理審査の予定

#### A-4: チームの開発力の証明
**現状**: 3名の技術力が不透明（特にメタバース開発）
**修正案**:
- 高崎：WAIS実施経験XX件、開発実績（GitHub URL等）
- 山﨑：Kitazunaでの開発実績、使用言語・フレームワーク
- 井上：数理的なスキル、プログラミング実績
- **Unity/3D開発は外注 or メンターに依頼** と明記（正直さが評価される）
